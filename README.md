# Quad Solutions - Medical Credentialing Website

## Project Overview
A professional corporate website for a medical credentialing company built with HTML, CSS, and JavaScript. The website showcases credentialing services, builds trust through testimonials, and captures leads through a contact form.

## Tech Stack
- HTML5
- CSS3 (Custom CSS)
- Vanilla JavaScript
- Font Awesome Icons
- Google Fonts (Inter)
